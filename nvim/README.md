# Neovim

Install neovim: `sudo apt install neovim`

Dependencies for plugins:
- For telescope: `sudo apt install ripgrep`

## Plugins

### treesitter
### telescope
### undotree
### lsp-zero
	- lsp
	- cmp (autocomplete)
	- mason (installing lsp language servers)
### nvim-tree
This overrides the default "netrw", so make sure to disable netrw.
